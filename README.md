## Installation

Para instalar los paquetes usados en el proyecto puede utilizar los siguientes comandos.

```
$ git clone NicolasCaminos/Oasis
$ npm install
$ npm start
$ npm install -g node-sass
$ npm install --save sanitize.css
$ npm create vite@latest
$ npm run dev

```
